Why IdentyClaw

Identity infrastructure for AI agents that must prove itself—consistently, portably, and on terms the holder controls.

🎫

Mint once, recognize everywhere

One on-chain Passport replaces repeated vendor onboarding. Any service that implements Passport recognition can verify you without OAuth tokens or API keys.

🔑

Sovereign ownership

Your NEAR account holds and controls the Passport. Keys stay with you—not in a central identity database.

🤝

HOLA mutual authentication

Prove identity to peer agents with NOISE framework secured cryptographic handshakes—directly on-chain or via API verify helpers.

🤖

Built for autonomous agents

Move from cumbesome API keys, and third pary OAth tokens, toward persistent, accountable actors that collaborate across services and channels.

How to Enroll

Four steps. The agent does setup and API login; you buy the Passport once in the browser.

  1. 1

    Agent

    Install gennearaccount

    Install the CLI on the agent host (enrollment guide).

  2. 2

    Agent

    Generate a NEAR account

    Run gennearaccount and store the key JSON securely, then share the publc key (NEAR account ID) with the user.

  3. 3

    You

    Purchase your Passport

    At purchase.identyclaw.com, mint with your NEAR account ID and pick role and traits like your unique facial features .

  4. 4

    Agent

    Login and use the API

    Sign a login timestamp at api.identyclaw.com for a JWT, then use identity to collaborate securely with other agents on demand and at scale.

Powered by RODiT

IdentyClaw Passports are built on RODiT (Rich Online Digital Tokens)—the underlying authentication technology on NEAR. Features of the stack:

🧩

Unified credential

Authentication, configuration, and licensing in one on-chain token.

🔐

Mutual authentication

Default mutual auth for clients, servers, and webhooks with strong MITM resistance.

⚙️

Stateless enforcement

Permissions, rate limits, and policy checks without backend session state.

🔑

Local keys & rotation

Keys never leave the endpoints; can be rotated idependently as often as you like by transferring your IdentyClaw Passport to a new NEAR account.
